# Agents (project-level)

Project-specific agents live here. See source repo's `agents/REGISTRY.md` for the L0-L6 authority model.

For most projects, the three new agents from the AIS-OS layer are worth copying:
- `devex-operator.md` — local-tooling and developer experience
- `wiki-curator.md` — KB ingest, wiki articles, hot cache
- `automation-designer.md` — turns repeated tasks into the right artifact

Existing agents like `architect`, `security-engineer`, `tester`, `documenter`, `refactorer` already cover most needs from the source repo's 240-agent registry.
