Run this exact output format:

1. **Raw Ideas** — 5–8 niche ideas in 60 seconds
2. **Validation Score** — pass/fail + why (4-step check below)
3. **MVP Build Plan** — 3–7 day timeline + exact tool prompts
4. **Launch & Marketing Funnel** — full 30-day attack plan

## Step 1: Validate (ALWAYS RUN FIRST — <1 hour)

### Pre-validated niches (proven people pay)
- Productivity / focus / habit trackers
- Health / fitness / wellness
- Finance / budgeting / investing
- Dating / social / networking
- Learning / education / language
- Entertainment / games / utilities
- Niche lifestyle (parenting, pet, travel)

### 4-Step Validation
1. **Painful daily problem** solvable by an app
2. **App Store Check**: 3+ apps making >$10k/mo (Sensor Tower or App Store charts)
3. **Competitor Deep Dive**: Download top 3, screenshot onboarding + core loop + paywall
4. **TikTok/IG Proof**: Must see influencers + faceless slideshows + UGC + paid ads

→ ALL 4 pass = **VALIDATED**. Any fail = kill and pick new niche.

---

## Step 2: Build MVP (3–7 Days)

**Tool stack**: Rork + Cursor (build) → ChatGPT (design prompts) → Superwall/RevenueCat (paywall) → Xcode (launch)

**Onboarding = 70% of revenue**:
1. Remind user why they downloaded
2. Make them feel the pain
3. Show app as perfect solution
4. End with **unskippable hard paywall** (free trial or upfront)

Copy onboarding structure exactly from best competitor. Make core app original. Ugly is fine. Launch in 3–7 days max.

**Pricing**: Mirror top competitors exactly.

---

## Step 3: Launch

- Apple Developer Account: $99/year
- Assets: Clear name + benefit-focused subtitle + 3–5 screenshots (big text, one idea/screen) + simple description
- Winner subtitle format: "[Action] and [benefit]" — e.g. "Block distractions and stay focused longer"
