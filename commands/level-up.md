# /level-up — Weekly Improvement Loop

Invokes `skills/level-up/SKILL.md`.

## What this does
Five questions about your week. Output: ONE recommended next artifact (skill, script, command, recipe, connection, or cadence change) with a small POC plan. Logged to `decisions/log.md`.

## What this does NOT do
- Does not build the artifact (separate session)
- Does not produce a backlog (one recommendation only — Bike Method)
- Does not auto-schedule recurring jobs

## Pairing
- Run after `/audit` — the audit's top gap is fed in as input
- Friday weekly cadence

## Output
- A single proposed artifact + plan in chat
- Entry in `decisions/log.md`
