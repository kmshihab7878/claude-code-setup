# B2C App Playbook — Ernesto Lopez Framework

Source: @ErnestoSOFTWARE — 12 iOS apps, $73k MRR. First app Snapout: $20k in 30 days from 2 viral videos.

**Core belief**: Apps are digital real estate. Build once → recurring revenue → no fulfillment. MVP ugly is fine. Speed > perfection.

---

## Step 1: Validate (Never Skip — <1 Hour)

**Rule**: Creativity is a double-edged sword. Never build a "dream app" nobody wants.

### Pre-Validated Niches (proven people pay)
- Productivity / focus / habit trackers
- Health / fitness / wellness
- Finance / budgeting / investing
- Dating / social / networking
- Learning / education / language
- Entertainment / games / utilities
- Niche lifestyle (parenting, pet, travel, etc.)

### 4-Step Validation Process (run in sequence)

**1. Find a painful daily problem** solvable by an app.

**2. App Store Check**: Search the niche. Must see 3+ apps making >$10k/mo (use Sensor Tower or App Store top charts).

**3. Competitor Deep Dive**: Download top 3 money-makers. Screenshot: onboarding flow + core loop + paywall. Study every detail.

**4. TikTok/IG Proof**: Search the niche. Must see:
- Influencers
- Faceless slideshows
- UGC creators
- Paid ads

→ If all 4 exist + competitors make money = **IDEA VALIDATED**. Move to build.

**Kill signal**: If 3+ apps NOT making >$10k OR no active TikTok/IG content → kill the idea, pick another niche.

## Step 2: Build MVP (3–7 Days)

### Tool Stack (2026 meta)
| Tool | Purpose |
|------|---------|
| Rork + Cursor | AI coding the entire app |
| ChatGPT | Prompt engineering for design |
| Superwall or RevenueCat | Paywalls |
| Firebase | Database (only if needed) |
| Pinterest + Dribbble | Design inspiration |
| Xcode | Final launch |

### Build Process

**1. Design**: Save 5–10 screenshots from Pinterest/Dribbble + top competitors. Prompt ChatGPT to turn reference images into a clean iOS UI prompt. Feed into Rork/Cursor.

**2. Onboarding = 70% of revenue**. Copy structure exactly from best competitor (proven conversion). Optimal flow:
1. Remind user why they downloaded
2. Make them feel the pain
3. Show app as perfect solution
4. End with **unskippable hard paywall** (free trial or upfront)

Make core app original. Onboarding = validated data from competitors.

**3. Pricing**: Mirror top competitors exactly.

**4. MVP Rule**: Ugly is fine. Launch in 3–7 days max.

---

## Step 3: Launch

- Apple Developer Account: $99/year (apply immediately, 3–5 days approval)
- **Minimum assets**: Clear app name + subtitle (benefit-focused), 3–5 screenshots (big text, one idea per screen), simple description
- Screenshot style: Cal AI ($3.6M/mo) — minimal UI, huge benefit text

**Winner subtitle format**: "Block distractions and stay focused longer" (not vague AI fluff)

---

## Step 4: Market — The 5 Channels (Run All Simultaneously)

| Channel | Cost | Execution | Notes |
|---------|------|-----------|-------|
| **UGC Creators** | Paid | Hire creators for 30–60 videos/mo | $15/video + bonuses ($150 at 100K views, $800 at 1M). Find via TikTok DMs. Sign contracts. |
| **Influencers** | Paid | $1 CPM deals only | One viral video = $20k (exact first-app case). Cap at $1k CPM. |
| **Faceless Content** | Free | 3–5 accounts, 3–5x/day (slideshows, screen recordings, text-on-screen) | Consistent style/character only. One hit → double down on exact format. |
| **Founder-Led** | Free | Film yourself using viral formats | Develop "viral sense" to feed creators. |
| **Paid Ads** | Paid | Meta + TikTok (use Singular for attribution) | Most scalable. Find winning creatives → infinite scale. |

**Key rules**:
- Track everything
- If one format hits → copy it across all creators
- One viral = game over
- Stay consistent (faceless accounts must look like a brand, not a random Pinterest feed)

---

## Step 5: Scale

- **Passive nature**: Most apps require zero maintenance after launch
- **Portfolio strategy**: Build 10+ apps, treat as digital real estate
- **Recurring revenue**: Subscriptions + hard paywalls = compounding wealth
- **Repeat formula**: Once you crack one niche, rinse and repeat
