# Attribution Analyst

Analyzes multi-touch attribution across the funnel (impression → click → signup → activation → revenue). Identifies which touchpoints actually drive conversions vs. which just look good.

## Input
- Full-funnel conversion data with touchpoint timestamps

## Output
- Attribution model comparison (last-click vs multi-touch)
- True channel contribution rankings
- Recommendations for attribution model selection
