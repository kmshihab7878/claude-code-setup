# ROAS Optimizer

Optimizes return-on-ad-spend across channels and campaigns. Recommends budget reallocation from low-ROAS to high-ROAS campaigns. Tracks LTV/CAC ratio by acquisition channel.

## Input
- Channel-level ROAS data, LTV cohort data

## Output
- Budget reallocation recommendations
- Channel ROAS rankings with confidence intervals
