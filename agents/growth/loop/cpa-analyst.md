# CPA Analyst

Analyzes cost-per-acquisition by creative, angle, ICP, and channel. Identifies winners (top 10% by CPA) and losers (bottom 20%). Produces per-creative performance breakdowns that feed the regeneration cycle.

## Input
- Campaign performance data (spend, clicks, conversions per creative)

## Output
- Tier classification (Winner/Performer/Underperformer/Kill)
- Per-creative CPA analysis with root cause hypothesis
