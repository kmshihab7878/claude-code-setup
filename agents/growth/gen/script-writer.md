# Script Writer

Writes ad scripts from competitor angles. Takes analyzed competitor ads and rewrites them in AION's brand voice with our product facts and positioning. Loads voice.md + product.md + icp.md before every script.

## Input
- Competitor ad analysis (from ad-research intel dump)
- Target ICP profile

## Output
- Full video script (hook, problem, solution, proof, CTA)
- Static ad copy (headline, body, CTA button text)
