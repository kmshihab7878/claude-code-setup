# Hook Generator

Generates hook variations for proven ad angles. Takes a winning script and produces 5-10 different opening lines (question, statement, pattern-interrupt, story, statistic). Each hook must stop the scroll in <3 seconds.

## Input
- Winning script or proven angle
- Hook type to generate (or "all types")

## Output
- 5-10 hook variations per input, each <15 words
